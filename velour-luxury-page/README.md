# Velour — Luxury Product Landing Page

A premium product page for a luxury perfume brand, built as a single-file web application. Demonstrates high-end web design with attention to typography, spacing, color theory, and animation.

## Features

- Premium serif and sans-serif typography pairing (Cormorant Garamond + Outfit)
- Dark luxury color palette with gold accent system
- Scroll-triggered animations and visual transitions
- Responsive layout from desktop to mobile
- Product showcase with ingredient storytelling
- Custom CSS design tokens for consistent styling

## Tech Stack

- HTML5
- CSS3 (custom properties, advanced animations, backdrop filters)
- JavaScript (scroll observers, interaction handlers)
- Google Fonts

## Design Approach

The page was designed to feel like a luxury magazine editorial rather than a typical product page. Every visual decision — the dark background, the gold accents, the serif typography, the generous whitespace — was chosen to convey exclusivity and craftsmanship.

## How It Was Built

Built using AI-assisted development with Claude Code. The design direction, copy, and visual system were developed through iterative refinement focused on achieving a premium aesthetic that competes with real luxury brand websites.

## Preview

Open `index.html` in any modern browser. Best viewed at full screen width for the intended visual experience.
