# Jebril Osawaru — Project Portfolio

A collection of working projects built by directing AI tools, not by traditional coding.

I am not a programmer and I do not write code by hand. Each project here started as a plain-English description of what I wanted. I built it by working closely with an AI assistant (Claude Code): directing it clearly, reviewing what it produced, testing the result, and correcting it until the project actually worked.

The skill on display is not programming. It is knowing how to direct an AI well, judge the quality of what it returns, and push it until the output is right.

## Projects

### [Fitness Dashboard](./fitness-dashboard/)
An interactive health and fitness tracking dashboard with real-time data visualization, a dark and light mode toggle, and animated interface components. Built in HTML, CSS, JavaScript, and Plotly.js.

### [Velour Luxury Product Page](./velour-luxury-page/)
A high-end product landing page for a luxury perfume brand. Premium typography, scroll-triggered animations, and responsive design, all in a single HTML file.

### [PPTX Animation Tool](./pptx-animation-tool/)
A Python tool that adds click-to-reveal animations to PowerPoint presentations. It works around a limitation in the python-pptx library by directly editing the underlying animation XML.

### [AI Image Generator](./ai-image-generator/)
Python scripts that connect to the Kie.ai API to generate realistic images from structured prompts. Supports image generation, editing, and image-to-video conversion.

### [Claude Code Complete Setup Guide](./claude-code-setup-guide/)
A 47-page documentation guide that takes a fresh VS Code and Claude Code installation from zero to a fully working AI-assisted environment. Covers setup, skills, knowledge files, and workflow.
