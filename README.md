# Jebril Osawaru — Project Portfolio

A collection of projects built using AI-assisted development (Claude Code) and traditional programming. Each project demonstrates practical problem-solving across web development, automation, and AI integration.

## Projects

### [Fitness Dashboard](./fitness-dashboard/)
Interactive health and fitness tracking dashboard with real-time data visualization, dark/light mode toggle, and animated UI components. Built with HTML, CSS, JavaScript, and Plotly.js.

### [Velour Luxury Product Page](./velour-luxury-page/)
High-end product landing page for a luxury perfume brand. Features premium typography, scroll-triggered animations, and responsive design built entirely in a single HTML file.

### [PPTX Animation Tool](./pptx-animation-tool/)
Python tool that injects click-to-reveal animations into PowerPoint presentations. Solves a limitation in python-pptx by directly manipulating OOXML animation XML.

### [AI Image Generator](./ai-image-generator/)
Python scripts that interface with the Kie.ai API to generate hyper-realistic AI images from structured JSON prompts. Supports image generation, editing, and image-to-video conversion.
