# Fitness Dashboard

An interactive fitness and health tracking dashboard with real-time data visualization, built as a single-file web application.

## Features

- Interactive charts and graphs powered by Plotly.js (weight tracking, workout frequency, calorie intake, body composition)
- Dark and light mode toggle with smooth transitions
- Animated background elements and floating orbs for visual depth
- Responsive layout that adapts to desktop, tablet, and mobile screens
- Custom CSS design system with CSS variables for theming
- Clean data display with progress indicators and stat cards

## Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, grid, animations)
- JavaScript (vanilla, no frameworks)
- Plotly.js for data visualization

## How It Was Built

Built using AI-assisted development with Claude Code. The design, layout, interactivity, and data visualization were developed iteratively through prompt engineering and code refinement. The entire application is contained in a single HTML file for easy deployment and sharing.

## Preview

Open `index.html` in any modern browser to view the dashboard.
